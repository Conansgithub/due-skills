# 网格服务器 (Mesh)

## 概述

网格服务器（mesh）即微服务，是 Due 框架集群架构中的无状态业务处理单元。支持 GRPC 和 RPCX 两种主流RPC框架，用于构建分布式微服务架构。

## 包路径

```go
import "github.com/dobyte/due/v2/cluster/mesh"
```

## 配置参数

```toml
[cluster.mesh]
    # 实例ID，集群中唯一。不填写默认自动生成唯一的实例ID
    id = ""
    # 实例名称
    name = "mesh"
    # 是否将内部通信地址暴露到公网。默认为false
    expose = false
    # 编解码器。可选：json | proto。默认为proto
    codec = "proto"
    # 实例元数据
    [cluster.mesh.metadata]
        # 键值对，且均为字符串类型
        key = "value"

[transport.grpc.server]
    # 服务器监听地址。空或:0时系统将会随机端口号
    addr = ":0"
    # 是否将内部通信地址暴露到公网。默认为false
    expose = false
    # 私钥文件
    keyFile = ""
    # 证书文件
    certFile = ""

[transport.grpc.client]
    # CA证书文件
    caFile = ""
    # 证书域名
    serverName = ""
```

## GRPC 示例

### 服务端

```go
package main

import (
    "context"
    "github.com/dobyte/due/registry/consul/v2"
    "github.com/dobyte/due/transport/grpc/v2"
    "github.com/dobyte/due/v2"
    "github.com/dobyte/due/v2/cluster/mesh"
)

type Server struct {
    pb.UnimplementedGreeterServer
    proxy *mesh.Proxy
}

func NewServer(proxy *mesh.Proxy) *Server {
    return &Server{proxy: proxy}
}

func (s *Server) Init() {
    s.proxy.AddServiceProvider("greeter", &pb.Greeter_ServiceDesc, s)
}

func (s *Server) Hello(ctx context.Context, args *pb.HelloArgs) (*pb.HelloReply, error) {
    return &pb.HelloReply{Message: "Hello " + args.Name}, nil
}

func main() {
    // 创建容器
    container := due.NewContainer()
    // 创建服务发现
    registry := consul.NewRegistry()
    // 创建RPC传输器
    transporter := grpc.NewTransporter()
    // 创建网格组件
    component := mesh.NewMesh(
        mesh.WithRegistry(registry),
        mesh.WithTransporter(transporter),
    )
    // 初始化应用
    initAPP(component.Proxy())
    // 添加网格组件
    container.Add(component)
    // 启动容器
    container.Serve()
}

func initAPP(proxy *mesh.Proxy) {
    server.NewServer(proxy).Init()
}
```

### 客户端

```go
package main

import (
    "context"
    "github.com/dobyte/due/registry/consul/v2"
    "github.com/dobyte/due/transport/grpc/v2"
    "github.com/dobyte/due/v2/log"
)

func main() {
    // 创建服务发现
    registry := consul.NewRegistry()
    // 构建传输器
    transporter := grpc.NewTransporter()
    // 设置默认的服务发现组件
    transporter.SetDefaultDiscovery(registry)

    // 构建客户端
    client, err := transporter.NewClient("discovery://greeter")
    if err != nil {
        log.Errorf("create rpc client failed: %v", err)
        return
    }

    // 获取GRPC客户端
    grpcClient := client.Client().(grpc.ClientConnInterface)
    
    // 创建服务客户端
    cli := pb.NewGreeterClient(grpcClient)

    // 发起RPC调用
    reply, err := cli.Hello(context.Background(), &pb.HelloArgs{Name: "fuxiao"})
    if err != nil {
        log.Errorf("invoke rpc func failed: %v", err)
        return
    }

    log.Infof("invoke rpc func replay: %v", reply)
}
```

## RPCX 示例

### 服务端

```go
package main

import (
    "context"
    "github.com/dobyte/due/registry/consul/v2"
    "github.com/dobyte/due/transport/rpcx/v2"
    "github.com/dobyte/due/v2"
    "github.com/dobyte/due/v2/cluster/mesh"
)

const (
    service     = "greeter"
    servicePath = "Greeter"
)

type Server struct {
    proxy *mesh.Proxy
}

func NewServer(proxy *mesh.Proxy) *Server {
    return &Server{proxy: proxy}
}

func (s *Server) Init() {
    s.proxy.AddServiceProvider(service, servicePath, s)
}

func (s *Server) Hello(ctx context.Context, args *pb.HelloArgs, reply *pb.HelloReply) error {
    reply.Message = "Hello " + args.Name
    return nil
}

func main() {
    container := due.NewContainer()
    registry := consul.NewRegistry()
    transporter := rpcx.NewTransporter()
    
    component := mesh.NewMesh(
        mesh.WithRegistry(registry),
        mesh.WithTransporter(transporter),
    )
    
    initAPP(component.Proxy())
    container.Add(component)
    container.Serve()
}

func initAPP(proxy *mesh.Proxy) {
    server.NewServer(proxy).Init()
}
```

### 客户端

```go
package main

import (
    "context"
    "github.com/dobyte/due/registry/consul/v2"
    "github.com/dobyte/due/transport/rpcx/v2"
    "github.com/dobyte/due/v2/log"
)

func main() {
    registry := consul.NewRegistry()
    transporter := rpcx.NewTransporter()
    transporter.SetDefaultDiscovery(registry)

    client, err := transporter.NewClient("discovery://greeter")
    if err != nil {
        log.Errorf("create rpc client failed: %v", err)
        return
    }

    rpcxClient := client.Client().(*rpcxClient.OneClient)
    cli := pb.NewGreeterOneClient(rpcxClient)

    reply := &pb.HelloReply{}
    err = cli.Hello(context.Background(), &pb.HelloArgs{Name: "fuxiao"}, reply)
    if err != nil {
        log.Errorf("invoke rpc func failed: %v", err)
        return
    }

    log.Infof("invoke rpc func replay: %v", reply)
}
```

## 最佳实践

1. **服务设计**:
   - 遵循单一职责原则
   - 合理划分服务边界

2. **协议选择**:
   - GRPC 适合高性能场景
   - RPCX 适合需要更多特性的场景

3. **服务发现**:
   - 使用 `discovery://` 前缀进行服务发现
   - 配合注册中心实现动态路由

4. **错误处理**:
   - 实现重试机制提高可用性
   - 设置合理的超时时间

5. **监控告警**:
   - 监控RPC调用延迟和成功率
   - 记录关键业务指标
