# 网关服务器 (Gate)

## 概述

网关服务器（gate）是 Due 框架集群架构的前端入口，负责处理客户端连接、协议解析、消息转发等核心功能。支持 TCP 和 WebSocket 协议，提供高性能的网关服务。

## 包路径

```go
import "github.com/dobyte/due/v2/cluster/gate"
```

## 使用示例

### 基础网关服务器

```go
package main

import (
    "github.com/dobyte/due/locate/redis/v2"
    "github.com/dobyte/due/network/tcp/v2"
    "github.com/dobyte/due/registry/consul/v2"
    "github.com/dobyte/due/v2"
    "github.com/dobyte/due/v2/cluster/gate"
)

func main() {
    // 创建容器
    container := due.NewContainer()
    // 创建用户定位器
    locator := redis.NewLocator()
    // 创建服务发现
    registry := consul.NewRegistry()
    // 创建网络服务器
    server := tcp.NewServer()
    // 创建网关组件
    component := gate.NewGate(
        gate.WithLocator(locator),
        gate.WithRegistry(registry),
    )
    // 初始化应用
    initApp(component.Proxy())
    // 添加网关组件
    container.Add(component)
    // 启动容器
    container.Serve()
}

// 初始化应用
func initApp(proxy *gate.Proxy) {
    // 监听连接事件
    proxy.AddEventListener(gate.Connect, connectHandler)
    // 监听断开连接事件
    proxy.AddEventListener(gate.Disconnect, disconnectHandler)
}

// 连接建立处理器
func connectHandler(conn *gate.Conn) {
    println("Client connected:", conn.ID())
}

// 连接断开处理器
func disconnectHandler(conn *gate.Conn) {
    println("Client disconnected:", conn.ID())
}
```

### 消息转发

```go
package main

import (
    "github.com/dobyte/due/v2/cluster"
    "github.com/dobyte/due/v2/cluster/gate"
)

func main() {
    component := gate.NewGate()
    proxy := component.Proxy()
    
    // 监听消息事件
    proxy.AddEventListener(gate.Receive, func(ctx *gate.Context) {
        // 解析消息
        msg := &cluster.Message{}
        if err := ctx.Parse(msg); err != nil {
            return
        }
        
        // 转发消息到节点服务器
        ctx.Forward(msg)
    })
}
```

## 最佳实践

1. **连接管理**:
   - 合理设置最大连接数防止资源耗尽
   - 启用心跳检测及时清理死连接

2. **安全性**:
   - 启用 TLS 加密保护通信
   - 使用加密器保护消息内容

3. **消息转发**:
   - 合理设计消息路由规则
   - 使用用户定位器追踪用户位置

4. **性能优化**:
   - 根据业务需求调整消息缓冲区大小
   - 使用异步处理提高吞吐量

5. **监控告警**:
   - 监听连接和断开事件统计在线人数
   - 监控消息转发延迟和成功率
